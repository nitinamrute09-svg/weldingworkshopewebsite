Admin Dashboard Disabled on Live Website

Because Vercel uses temporary, read-only servers, any images uploaded here would be permanently deleted after a few minutes.

How to upload images safely:

  1. Open this code project on your computer.
  2. Run python app.py in your terminal.
  3. Go to http://127.0.0.1:5000/admin in your browser.
  4. Log in and upload your images there!
  5. When finished, run git push in your terminal to save the images permanently to GitHub and this live website.
